## 准备系统

到Windows 11官网下就行，刻录到U盘里，制作启动盘。避免装不上win11还再下了个win10。结果win11直接就装上了。

## 装机

换掉硬盘插u盘，按F9进入启动盘，就直接安装了，十分钟搞定。一个小的坑是硬盘插了半天，和原本的卡接口不一样，怕不兼容，担心硬插插坏了。角度斜向下，用点力怼。

## 工作

- git：detected dubious ownership。原系统用户发生变化了。[解决](https://stackoverflow.com/questions/72978485/git-submodule-update-failed-with-fatal-detected-dubious-ownership-in-repositor)。
    
    ```jsx
    git config --global --add safe.directory '*'
    ```
    
    同时要连接github，重新生成一个ssh key上传。
    

## 后遗症恢复

### Adobe Acrobat

每次打开都问，分发的协议。烦死人。

重装后还找不到卸载，没提供uninstaller的家伙真讨厌。直接把安装整个文件夹删掉，然后用cclearner清楚注册表，保险起见自己再手动删除一遍。参考[here](https://blog.csdn.net/Jerry_liu20080504/article/details/100067333)，软件获取参考[here](http://www.ddooo.com/softdown/225409.htm)。

### Office

重装没自带的office了。懒得找破解版，好难找。直接去官网用试用订阅一个月下载office 365。下载完之后验证账户就可以重新用原本自带那个license了，不过就得变为office 2016.

### Navicat

重新破解一遍吧，把libcc.dll删了，恢复之前的lib.cc.bak。重新patch，重新注册。注册时断网。

### Git

将.gitconfig拷贝回去

	email = 861921648@qq.com
[http]

### ssh

将原本配置过的ssh/config拷贝回去

```jsx
Host ubuntu
    HostName 114.132.217.17
    Port 22
    User ubuntu
    IdentityFile ~/.ssh/ubuntu.pem
    ServerAliveInterval 30
    TCPKeepAlive yes

```
